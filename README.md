# Draw & Guess Game 🎨

Game multiplayer Draw & Guess - main bareng teman!

## Fitur

- **Multiplayer Real-time** - Sampai 30 pemain
- **Multi Tema** - Hewan, Makanan, Alam, Fantasi, Kendaraan, Benda
- **Tools Gambar** - Brush, Penghapus, Isi Warna, Clear
- **30+ Warna** - Palette warna lengkap + custom color picker
- **5 Ukuran Brush** - Dari tipis sampai tebal
- **Timer** - 80 detik per ronde
- **Sistem Hint** - Hint muncul saat waktu habis
- **Skor** - Poin berdasarkan kecepatan tebak
- **Leaderboard** - Lihat siapa pemenangnya
- **Chat** - Obrolan dan tebakan real-time
- **Responsive** - Bisa di HP dan Desktop
- **Dark Mode** - Nyaman di mata
- **Musik Latar** - Tambahkan musik sendiri
- **Session Save** - Refresh tidak kehilangan sesi
- **Bahasa Indonesia** - UI lengkap Bahasa Indonesia

## Cara Install

```bash
# Install dependencies
npm install

# Start server
npm start

# Buka browser
http://localhost:3000
```

## Deploy ke Replit

1. Buka replit.com
2. Create Repl → Import from GitHub
3. Pilih repo draw-guess-game
4. Run Command: `npm start`
5. Klik Run ▶️
6. Klik Publish

## Cara Main

1. **Buat Room** - Masukkan nama, klik "Buat Room"
2. **Share Kode** - Beri kode 4 huruf ke teman
3. **Gabung** - Teman masukkan kode untuk gabung
4. **Siap** - Semua klik "Siap!"
5. **Gambar** - Kalau giliranmu, pilih kata dan gambar
6. **Tebak** - Yang lain ketik tebakan di chat
7. **Skor** - Tebakan cepat = poin lebih banyak
8. **Menang** - Skor tertinggi menang!

## Tambah Musik Sendiri

1. Simpan file musik di folder `public/music/`
2. Rename jadi `bgm.mp3`
3. Musik akan diputar otomatis

Format yang didukung: MP3, OGG, WAV
