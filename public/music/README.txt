# Folder Musik

Simpan file musik kamu di folder ini.

## Cara Menambahkan Musik:

1. Rename file musik kamu menjadi: `bgm.mp3`
2. Paste file `bgm.mp3` ke folder ini
3. Musik akan otomatis diputar saat game dimulai

## Format yang Didukung:
- MP3 (.mp3)
- OGG (.ogg)
- WAV (.wav)

## Catatan:
- File harus bernama `bgm.mp3`
- Ukuran maksimal disarankan < 10MB
